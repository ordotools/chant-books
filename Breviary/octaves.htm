The following rules govern the commemorations of octaves in Vespers:

 

If the current day and the following day are both of the octave, the commemoration is from Second Vespers of the octave.

If neither the current day nor the following day are of the octave, the commemoration is from Second Vespers of the octave.

If the current day is of the octave but the following day is not, the commemoration is from Second Vespers.

If the current day is not of the octave, but the following day is, the commemoration is from First Vespers.

 

An octave is commemorated in Vespers using the antiphon from Second Vespers in all cases except when the following day is of the octave day, or the current day is not of the octave and the following day is, even if the octave day is overridden by another feast.
